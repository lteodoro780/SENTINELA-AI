# SENTINELA AI

Offline AI environment focused on enterprise and laboratory deployments using open-source technologies.

## Overview

SENTINELA AI is a self-hosted artificial intelligence environment designed for offline or restricted networks.

The project focuses on:

* Local AI inference
* Document-based RAG environments
* Enterprise-friendly deployment
* Docker-based infrastructure
* Low-cost hardware compatibility
* Secure offline environments

## Features

* Offline AI operation
* Docker deployment
* Open WebUI integration
* Ollama integration
* Local document processing
* LAN-only operation
* Lightweight infrastructure support

## Technologies

* Docker
* Ollama
* Open WebUI
* Linux
* Bash
* Self-hosted infrastructure

## Project Structure

```text
docs/       Documentation
scripts/    Automation scripts
configs/    Configuration files
images/     Screenshots and diagrams
```

## Deployment

The environment can be deployed using Docker Compose on Debian-based systems.

## Goals

* Provide a fully offline AI environment
* Support restricted enterprise networks
* Reduce dependency on cloud services
* Simplify AI deployment for internal environments
